# Notation


| Notation          |  Semantics                               |
|-------------------|------------------------------------------|
| `x`               | literally the character(s) x             |
|  Îµ                | the empty string                         |
| <*t*><sup>?</sup> | optional                                 |
| <*t*><sup>*</sup> | zero or more                             |
| <*t*><sup>+</sup> | one or more                              |
| <*t*> \| <*u*>     | either <*t*> or <*u*>                              |
