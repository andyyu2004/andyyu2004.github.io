L

Introduction

L Programs

Syntax

<program> ::= <item>*

An L program consists of a collection of items where one of them is the main function.

Lexical Syntax

  • struct
  • enum
  • if
  • else

  • false
  • let
  • pub

Identifiers

Lexer:

[_a-zA-Z][a-zA-Z0-9]*

Comments

Lexer:

line-comment: //

block-comment : /* .. */

Items

Syntax:

<item> ::= <vis> <item-kind>

<item-kind>
    ::= <function>
      | <struct-decl>

Functions

Syntax:

<fn> ::= fn <ident> <generics>? (<params>?) <return-type> <block-expr>

<params> ::= <param> (, <param>)*

<param> ::= <expr> : <type>

<return-type> ::= -> <type>

Examples

fn identity<T>(t: T) -> T {
    t
}

fn naive_fib(n: number) -> number {
    if n < 2 { n } else { naive_fib(n-1) + naive_fib(n-2) }
}
